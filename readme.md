# HP Roller

:heart::game_die: An HP dice roller for D&D 5e

## Prerequisites

- :factory: .NodeJS v16

## Installation

1. :arrow_down_small: Download the repo
1. `nvm use 16.20.2 && npm i && npm run start`
1. Try it out!
   `    (index)   | HP Rolled | Initiative
    ----------------------------------
    ----------------------------------
        0     |   14      |    18
    ----------------------------------
        1     |   14      |    12
    ----------------------------------
   `
